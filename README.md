# 🐍 Snake Game

A simple and fun Snake Game built using **HTML, CSS, and JavaScript**.
Control the snake, eat food, and try to get the highest score without hitting the walls or yourself.

---

## 🚀 Features

* 🎮 Smooth snake movement
* 🍎 Random food generation
* 📈 Score tracking system
* 💀 Game over detection

---

## 🕹️ Controls

| Key                    | Action     |
| ---------------------- | ---------- |
| ⬆️ Arrow Up,     KeyW  | Move Up    |
| ⬇️ Arrow Down,   KeyS  | Move Down  |
| ⬅️ Arrow Left,   KeyA  | Move Left  |
| ➡️ Arrow Right,  KeyD  | Move Right |
